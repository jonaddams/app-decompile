   iOS APP SDK DETECTION - QUICK START FOR NON-TECHNICAL USERS
═══════════════════════════════════════════════════════════════

🎯 WHAT THIS DOES:
Checks if a former customer's app still has your SDK installed.

📋 WHAT YOU NEED:
1. A Mac computer
2. Your Apple ID (the one you use for App Store)
3. The customer's app URL from the App Store

═══════════════════════════════════════════════════════════════

⚡ QUICK START (3 STEPS):

Step 1: Open Terminal
  • Open Finder
  • Go to Applications → Utilities → Terminal
  • Double-click Terminal

Step 2: Navigate to the folder
  • Type: cd 
  • Drag the folder with these files into Terminal
  • Press Enter

Step 3: Run the check
  • Copy this command:
    chmod +x detect-sdk-{platform}.sh && ./detect-sdk-{platform}.sh -s pspdfkit -s nutrient -u "PASTE_APP_URL_HERE"
  
  • Replace PASTE_APP_URL_HERE with the App Store link
  • Press Enter

═══════════════════════════════════════════════════════════════

🎬 WHAT HAPPENS NEXT:

First Time Only:
  1. Script installs Homebrew (enter your Mac password when asked)
  2. Script installs ipatool
  3. Script asks for your Apple ID email
  4. You enter your Apple ID password
  5. You enter the 2FA code from your iPhone

Every Time:
  1. Downloads the app (2-5 minutes)
  2. Analyzes the app
  3. Shows result: SDK FOUND ✅ or NOT FOUND ❌
  4. Saves detailed report

═══════════════════════════════════════════════════════════════

📊 UNDERSTANDING RESULTS:

✅ SDK FOUND = Customer still has your SDK (compliance issue)
❌ SDK NOT FOUND = Customer removed it (all clear)

The report file is automatically named with:
  sdk-detection-{app-name}-{date-time}.txt

═══════════════════════════════════════════════════════════════

💡 EXAMPLES:

Check Data Collect Mobile app:
  ./detect-sdk-{platform}.sh -s pspdfkit -s nutrient -u "https://apps.apple.com/us/app/data-collect-mobile/id1494756647"

Check multiple customers:
  ./detect-sdk-{platform}.sh -s pspdfkit -b com.customer1.app
  ./detect-sdk-{platform}.sh -s pspdfkit -b com.customer2.app
  ./detect-sdk-{platform}.sh -s pspdfkit -b com.customer3.app

═══════════════════════════════════════════════════════════════

❓ COMMON ISSUES:

"Permission denied"
  → Run: chmod +x detect-sdk-{platform}.sh

"Command not found"  
  → Make sure you're in the right folder (Step 2)

"Authentication failed"
  → Check your Apple ID password
  → Make sure 2FA code hasn't expired
  → Try again

Script asks for password
  → This is normal! It needs it to install Homebrew
  → Enter your Mac login password
